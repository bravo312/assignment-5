How to Apply CSS to HTML

To include an external CSS file in an HTML document, use the <link> element within the <head> section. Here is an example:
<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the <script> element right before the closing </body> tag. Example:
<script src="scripts.js"></script>
To write JavaScript directly within an HTML file, use the <script> element. Here's an example: 

    <script>
    console.log('Hello, world!');
    </script>
